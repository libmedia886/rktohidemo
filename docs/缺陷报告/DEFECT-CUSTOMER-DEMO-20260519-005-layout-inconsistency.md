# DEFECT-CUSTOMER-DEMO-20260519-005: 页面风格和位置不一致

状态: In Progress
分类: Product Defect
范围: 默认轮播

## 问题描述

不同页面的标题、主画面、说明区、指标区位置差异较大，看起来不像同一套客户演示产品。

## 大概解决方案

定义统一页面模板：顶部标题和状态，中部主画面，底部流程图、中文说明和指标。逐页迁移到同一坐标和字号体系，只保留模块内容差异。

## 本轮进展

已优先整改 VMIX、OSD、RGA、CSC_CL 四个被点名页面的说明区位置、流程图表达和显示比例。其余默认轮播页仍需按同一模板继续收敛。
